\section{Problem}
Which of the following cannot be the probability of an event?
\begin{enumerate}[label={\roman*)}]
    \item $\frac{2}{3}$
    \item $-1.5$
    \item $15$
\end{enumerate}
\section{solution}
Let A be any event of any experiment then 
\begin{align}
    0 \leq P(A) \leq 1
\end{align}

    (i) $0 \leq \frac{2}{3} \leq 1$

     
\textbf{example}:\\
A bag contains one green ball,two blue balls.Probability of picking a blue ball from the bag is $\frac{2}{3}$\\
So probability of an event can be $\frac{2}{3}$\\

\vspace{1mm} %1mm vertical space
 (ii)$-1.5 < 0$\\
 it does not satisfy the condition (2.0.1)\\
 So probability of an event cannot be $-1.5$\\
 
 \vspace{1mm} %1mm vertical space
 (iii)$1 < 15$\\
 it does not satisfy the condition (2.0.1)\\
 So probability of an event cannot be $15$\\
 
 Therefore options (ii) and (iii) are the required answers.
